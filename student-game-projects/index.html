Creative Background

学生時代のゲーム制作

学生時代には、ゲームを遊ぶだけでなく、自分で作ることにも強く関心がありました。 規模は大きくありませんが、複数の技術やライブラリを使って実際にゲームを形にしていました。

今でも将来的にゲーム制作へ戻りたい気持ちがあり、その源泉の一つがこの時期の経験です。

Projects

作ったもの

curses で `ヘルメット`

端末ベースの描画と入力処理でゲームを組み立てる体験として印象に残っています。

Java の Swing でシューティングゲーム

GUI ベースで動くゲームを自分で作る感覚を掴んだ、初期の制作物の一つでした。

DX ライブラリでボンバーマン

2D ゲームらしい画面構成やルール実装を、自分の手で形にした題材です。

OpenGL で 3D ぷよぷよ

描画表現や立体的な見せ方まで踏み込んで触れた制作物でした。

Meaning

今につながっていること

  • 既存のものを使うだけでなく、自分で仕組みを作ってみたい志向
  • 技術を覚えるとき、完成物まで持っていく方が面白いという感覚
  • 将来、AI を活かした創作やゲーム制作に戻りたいという関心の起点
